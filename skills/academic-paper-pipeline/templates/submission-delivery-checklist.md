# Submission Delivery Checklist

## Manuscript

- Title matches contribution.
- Abstract matches actual results.
- Introduction names gap and contribution.
- Methods match data and analysis.
- Results match tables and figures.
- Discussion does not overclaim.
- Limitations are explicit.
- All abbreviations are defined.

## Evidence

- Every major claim has a citation or result.
- No invented citations.
- Tables and figures are cited in order.
- Statistical tests and models are named.
- Sensitivity/robustness checks are summarized.
- Data restrictions and ethics are disclosed.

## Figures And Tables

- Figures use journal-ready export settings.
- Axes include units.
- Legends are readable.
- Colors are accessible.
- Resolution is sufficient.
- Tables have clear notes.
- Supplementary tables are named consistently.

## Reproducibility

- Code run order is documented.
- Data paths are stable or documented.
- Software versions are recorded.
- Random seeds are set where relevant.
- Restricted data instructions are included.
- Output files can be regenerated or limitations are stated.

- Slide narrative matches paper contribution.
- Methods slide is understandable.
- Main results have no clutter.
- Limitations are prepared.
