# Figure Review Checklist

## Before Export

- Figure type matches the claim.
- Variables and units are correct.
- Groups are labeled clearly.
- Statistical test or model is named.
- Colors work in grayscale or with color-vision deficiency.
- Legend does not cover important data.
- Raw data are shown for group comparisons where feasible.
- Outliers are visible or explicitly handled.

## Style

- Arial or Helvetica.
- Base font about 11 pt.
- White background.
- No default gray ggplot panel.
- No unnecessary grid.
- Axis text is readable at final size.
- Panel tags are consistent.

## Export

- TIFF, EPS, or PDF as required.
- 300 DPI or higher for raster.
- Single-column or double-column width selected intentionally.
- File path is reproducible.
- Figure generated by code, not manual GUI export.

## Reviewer-Proofing

- Axis labels include units.
- P-value stars have a named test.
- Multiple comparisons are corrected or justified.
- Heatmap scaling is stated.
- Volcano thresholds are stated.
- KM event/censoring logic is stated.
- Forest plot effect metric is stated.
- Caption can be written from the plot code.
