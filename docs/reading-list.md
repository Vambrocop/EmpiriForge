# Reading List

This reading list is intentionally short. EmpiriForge should stay useful as a workflow library, not become a giant bibliography.

## AI-Assisted Reproducibility

- *Scaling Reproducibility: An AI-Assisted Workflow for Large-Scale Replication and Reanalysis*.
- Local Chinese translation and notes:
  - `docs/Scaling_Reproducibility_中文翻译.md`
  - `docs/Scaling_Reproducibility_中文解读.md`
  - `docs/Scaling_Reproducibility_可提炼skills.md`

## Applied Econometrics

- Angrist, Joshua D., and Jorn-Steffen Pischke. *Mostly Harmless Econometrics*.
- Angrist, Joshua D., and Jorn-Steffen Pischke. *Mastering 'Metrics*.
- Cunningham, Scott. *Causal Inference: The Mixtape*: https://mixtape.scunning.com/

## Causal Inference

- Hernan, Miguel A., and James M. Robins. *Causal Inference: What If*: https://www.hsph.harvard.edu/miguel-hernan/causal-inference-book/
- Imbens, Guido W., and Donald B. Rubin. *Causal Inference for Statistics, Social, and Biomedical Sciences*.
- Morgan, Stephen L., and Christopher Winship. *Counterfactuals and Causal Inference*.

## Causal Machine Learning

- Zhao and Liu, *Causal ML: Python package for causal inference machine learning*: https://doi.org/10.1016/j.softx.2022.101294
- CausalML documentation: https://causalml.readthedocs.io/
- CausalML GitHub repository: https://github.com/uber/causalml
- Künzel, Sekhon, Bickel, and Yu, meta-learners for estimating heterogeneous treatment effects.
- Nie and Wager, quasi-oracle estimation of heterogeneous treatment effects.

## How To Expand

Add a source only when it changes a workflow decision. A good source should answer at least one of these:

- What assumption must be named?
- What diagnostic should be run?
- What should a replication package contain?
- What should be disclosed to readers?
- What should an AI agent never infer without evidence?
